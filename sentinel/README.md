# 🛡️ Sentinel Protocol: Autonomous Solana DeFi Agent

**Sentinel Protocol** is a high-performance, autonomous trading agent designed for the Solana ecosystem. It merges the cognitive power of Large Language Models (LLMs) with a rigorous, multi-layered security engine to protect capital while executing elite on-chain strategies.

---

## 🧠 System Architecture

The protocol is divided into three core pillars that operate as a "Brain-Shield-Gateway" triad:

### 1. The Cognitive Core (`nlp_handler.py`)
The central nervous system of the agent, powered by the **Maverick Llama 4** model via the OpenRouter API.
* **Intent Disambiguation:** Translates natural language into machine-readable JSON actions like BUY, SELL, CHECK, or CHAT.
* **Neural Sync Streamer:** Generates high-fidelity "Thoughts" for the terminal, categorizing logic into pathways like COGNITION, SECURITY, and EXECUTION.
* **Cognitive Memory:** Manages short-term state and session context to prevent the agent from "forgetting" the token or action under discussion.

### 2. The Safety Sentinel (`safety_sentinel.py`)
The "Final Decision Gate" and circuit breaker that vetoes dangerous transactions.
* **On-Chain Security:** Integrates with the RugCheck API to detect mint authority risks, high rug scores (default threshold 500), and "danger" levels.
* **Multi-Factor Risk Scoring:** Calculates total risk by weighting on-chain data (50%), macro indicators (25%), and social sentiment (25%).
* **Dynamic Vetoes:** Implements "Hard Vetoes" for compromised tokens and "Soft Vetoes" if the market environment or bearish technical chart patterns exceed the allowed risk threshold.
* **Portfolio Health Analysis:** Evaluates the health score of existing assets (0-100) by checking them against security reports concurrently.

### 3. The Sentinel Bridge (`server.py`)
The FastAPI gateway handling authentication, real-time communication, and wallet orchestration.
* **High-Grade Security:** Implements Argon2 password hashing with high memory costs (64MB) and Brevo-powered 2FA OTP verification.
* **WebSocket Neural Sync:** Streams internal agent thoughts to the user's terminal via a persistent WebSocket connection (`/ws/neural-sync`).
* **Dashboard Integration:** Provides endpoints for live portfolio tracking, liquid staking yield analysis (JitoSOL, mSOL, bSOL), and a detailed trade history ledger.

---

## 🚀 Key Features

* **Autonomous Reconnaissance:** Performs deep-scan reports on Solana tokens, including liquidity checks and anomaly detection for "Pump & Dump" patterns.
* **Capital Protection:** A mandatory safety sweep must be cleared before any execution instruction is generated by the AI.
* **Portfolio Intelligence:** Real-time pricing of SOL and SPL tokens using a Market Resolver integrated with the Solana executor.
* **2FA Protected Execution:** Ensures trade execution only occurs within an authenticated, 2FA-verified session using encrypted private keys.

## ⚙️ Configuration & Setup

### Environment Variables
Ensure your `.env` file includes the following keys for full protocol activation:

```env
OPENROUTER_API_KEY=your_key
JWT_SECRET_KEY=your_secure_secret
BREVO_API_KEY=your_brevo_key
SENDER_EMAIL=auth@yourdomain.fun
FRONTEND_URL=your_frontend_url

Installation
Initialize Database: The protocol uses SQLAlchemy for identity and trade history management.

Boot the Server: Use uvicorn src.api.server:app to launch the FastAPI bridge.

Connect Terminal: Use the WebSocket endpoint to receive the live agent thought stream.
